# 软件质量管理复习平台 (SQM Review)

软件质量管理课程的系统性复习网站，基于 Vue 3 + Vite 构建，涵盖课件学习、复习背诵、真题练习和模拟考试四大模块。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite 5** - 下一代前端构建工具
- **Vue Router 4** - 官方路由管理器
- **Pinia 2** - Vue 状态管理库
- **Sass** - CSS 预处理器
- **Lucide Icons** - 图标库

## 快速开始

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装与运行

```bash
# 进入项目目录
cd sqm-review

# 安装依赖
npm install

# 启动开发服务器 (默认 http://localhost:5173)
npm run dev

# 生产构建
npm run build

# 预览构建产物
npm run preview
```

### 部署

构建产物位于 `sqm-review/dist/` 目录，可部署至任意静态文件服务器：

```bash
# 构建产物部署到 dist/ 目录
npm run build

# 使用 Vite 预览构建结果
npm run preview

# 或使用其他静态服务器
npx serve sqm-review/dist
```

## 数据文件说明
